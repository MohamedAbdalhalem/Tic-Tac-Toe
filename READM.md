# Tic-Tac-Toe Game

## Project Idea

A classic two-player Tic-Tac-Toe game where players take turns marking spaces on a 3×3 grid. The first player to get three marks in a row (horizontal, vertical, or diagonal) wins the game.

## Features

- Two-player gameplay
- Game state tracking
- Winning detection
- Move history log
- Reset/replay functionality

## Languages & Technologies

- **React 19.0** - JavaScript UI framework
- **Vite 4.4** - Build tool and development server
- **JavaScript (ES6+)** - Programming language
- **CSS** - Styling
